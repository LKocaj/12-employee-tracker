### Installation


* 1. Install Node Package Manager
  ```sh
  npm init
  ```
* 2. Install Inquirer
  ```sh
  npm install insquirer
  ```
* 3. Install mysql2
  ```sh
  npm install mysql
  ```
* 4. Install Console Table
  ```sh
  npm console.table
  ```
* 5. Run program:
  ```sh
  node index.js
  ```
